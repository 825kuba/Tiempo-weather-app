Tiempo weather is a single page weather app built with SCSS and JavaScript.

Bellow you can find a description of the app's features. For all the technical information and to see the code, please visit the project's GitHub page.

Tiempo allows you to:

  • Search
    Search for places and see the results in real time. Select one of the search results, use your position, or simply press enter to search for your query.
  • Switch between forecasts
    Switch between hourly and daily forecast on the weather card.
  • Check the time
    See what the local time is - each card has it's own working clock.
  • Add to favourites
    Add cards to favourites and see how many places you've added - the number updates in real time. Navigate between your favourite places using the buttons on screen, keyboard arrows on desktop or swipe gestures on touch screen.
  • Change settings
    Change time format, weather units and switch smooth scrolling on and off. Confirm changes by pressing the button, or cancel them by exiting the settings window.
